⚠️ BƯỚC QUAN TRỌNG NHẤT (Để tàng hình)

**Cơ chế tàng hình hoạt động bằng cách thay thế cơ thể bạn bằng ảnh nền phía sau. Do đó:**

1.  Mở web lên.
2.  **Đi ra khỏi khung hình camera** (để camera chỉ thấy ghế/tường trống).
3.  Nhấn phím **SPACE (Dấu cách)** để hệ thống chụp lại nền.

**Tàng hình**  **Nắm chặt tay TRÁI** (Giữ để duy trì). 
**Ném Flash**  **Nắm tay PHẢI** để tích tụ -> **Thả ra** để ném.
